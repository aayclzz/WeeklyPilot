# WeeklyPilot - 蓝桥周报自动生成系统 V4.1

> 可视化选择章节，AI 根据选择内容生成客观直白的周报。

**WeeklyPilot** 是一个智能周报生成助手，帮助蓝桥SaaS平台用户快速完成每周周报任务。

---

## 这个工具能干什么？

如果你在蓝桥 SaaS 平台需要**每周交周报**，这个工具可以帮你：

1. 根据课程ID获取完整的课程目录结构
2. 通过**可视化界面**选择需要写入周报的章节和小节
3. AI 根据选择的内容生成**客观、直白**的周报（无主观感受、不用"我"）
4. 自动填写到周报表单并提交

**V4.0 新特性**：
- ✅ 可视化章节选择界面
- ✅ 基于选择内容生成周报（不再瞎编）
- ✅ 直白客观的写作风格（像技术文档）
- ✅ 自动过滤"挑战"相关内容

## 你需要准备什么？

| 需要的东西 | 去哪里找 |
|-----------|---------|

## 快速上手（3 分钟）

### 第 1 步：安装 Python

如果电脑上还没有 Python，去 https://www.python.org/downloads/ 下载安装（需要 **Python 3.8** 及以上版本）。

> 装的时候记得勾上 **"Add Python to PATH"**（Windows）

### 第 2 步：运行程序

**Windows 用户**：
直接双击 `run.bat`，程序会自动完成以下操作：

1. 查找系统中的 Python（支持自定义安装路径、虚拟环境、py 启动器）
2. 检查 Python 版本是否符合要求
3. 如果没有 `.env` 配置文件，自动从模板创建并打开记事本让你填写
4. 如果缺少依赖包，自动使用国内镜像源安装

**macOS/Linux 用户**：
在终端中运行：
```bash
# 首次使用需要添加执行权限
chmod +x run.sh

# 运行程序
./run.sh
```

**首次运行时**，程序会弹出 `.env` 文件让你填写账号信息，只需要填 **3 个地方**：

```env
# ========== 蓝桥平台账号（必填） ==========
LANQIAO_USERNAME=13812345678      ← 改成你的手机号
LANQIAO_PASSWORD=你的密码          ← 改成你的密码

# ========== AI 密钥（必填） ==========
OPENAI_API_KEY=sk-xxxxxxxxxxxx    ← 改成你的 API Key
```

保存关闭后，再次双击 `run.bat` 即可使用。

## 使用流程

### 方式一：可视化选择（推荐）

1. 运行程序：
   - Windows: `run.bat` 或 `run.bat select`
   - macOS/Linux: `./run.sh` 或 `./run.sh select`

3. 浏览器自动打开选择界面

7. 确认内容后提交

1. 运行程序：
   - Windows: `run.bat direct`
   - macOS/Linux: `./run.sh direct`

3. 查看显示的章节列表
4. 输入要选择的编号（如 `1,3,5-8`）
5. AI 自动生成周报
6. 确认内容后提交

## 周报风格说明

V4.0 采用**直白客观**的写作风格：

- ✅ 只陈述事实，像技术文档
- ✅ 使用数字序号列出知识点
- ✅ 包含具体技术细节（命令、语法等）
- ❌ 不使用"我"、"我们"等第一人称代词
- ❌ 不使用"收获"、"感觉"、"体会到"等主观词汇
- ❌ 不提及"挑战"相关内容

**示例：**

```
【总结】
本周学习数据库技术课程，学习内容如下：
1. SELECT 基础查询语句，包含 WHERE 条件过滤
2. ORDER BY 排序语法，支持 ASC 升序和 DESC 降序
3. GROUP BY 分组查询，配合聚合函数使用

【所遇问题】
1. 多表连接查询的语法较复杂，容易混淆
2. 子查询的嵌套使用需要更多练习
3. 事务隔离级别的区别理解不够深入

【解决方案】
1. 通过编写多表连接查询语句进行实践练习
2. 查阅子查询相关资料，结合实例理解
3. 针对事务隔离级别进行专项练习

【下周计划】
周一：学习数据库的索引优化
周二：学习存储过程，复习周一所学内容
周三：学习触发器，复习周二所学内容
周四：学习视图，复习周三所学内容
周末：复习本周所学内容，预习下周课程
```

## 常见问题

### 如何获取课程ID？

打开蓝桥云课课程页面，URL中的数字就是课程ID。例如：
- `https://www.lanqiao.cn/courses/61384/` → 课程ID是 `61384`

### 报错"无法获取课程信息"？

如果看到类似以下错误：

```
ERROR | 无法获取课程 61384，61381 的信息
ERROR | 无法获取课程目录，请检查课程ID是否正确
```

**原因**：`.env` 文件中 `COURSE_IDS` 使用了**中文逗号** `，` 而不是英文逗号 `,`，导致程序把 `61384，61381` 当作一整个课程ID去请求，自然找不到。

**解决方法**：打开 `.env` 文件，检查逗号是否为英文逗号：

```env
# ❌ 错误（中文逗号）
COURSE_IDS=61384，61381
COURSE_NAMES=数据库技术，国产化操作系统1.8

# ✅ 正确（英文逗号）
COURSE_IDS=61384,61381
COURSE_NAMES=数据库技术,国产化操作系统1.8
```

> **提示**：程序已内置中文逗号自动转换，但建议仍然使用英文逗号以避免其他潜在问题。

### 浏览器选择界面没有打开？

检查是否有浏览器窗口被挡住。也可以手动打开浏览器访问终端显示的地址。

### 生成的周报内容不对？

1. 确认选择的章节是否正确
2. 检查AI API Key是否有效
3. 尝用测试模式测试：
   - Windows: `run.bat test 课程ID`
   - macOS/Linux: `./run.sh test 课程ID`

### 如何清除课程目录缓存？

```bash
# Windows
run.bat clear-cache

# macOS/Linux
./run.sh clear-cache
```

### API Key 无效？

- 检查 `.env` 里的 `OPENAI_API_KEY` 有没有写对
- 去 https://platform.deepseek.com/ 看看 API Key 有没有过期、余额够不够

## 技术原理

```
用户输入课程ID
    │
    ▼
course_catalog.py 获取课程目录
    │  调用蓝桥 API 获取完整目录结构
    │  缓存到 data/catalogs/
    │
    ▼
selection_server.py 启动选择界面
    │  打开浏览器显示目录树
    │  用户勾选章节/小节
    │  返回选择结果
    │
    ▼
ai_generator.py 生成周报
    │  根据选择的内容生成
    │  直白客观的风格
    │  自动过滤"挑战"相关内容
    │
    ▼
browser.py 填写并提交
    │  登录蓝桥平台
    │  填写周报表单
    │  等待用户确认后提交
```
